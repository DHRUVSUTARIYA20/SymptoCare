SYM PTOCARE - EASY SETUP (WINDOWS)

This file shows simple steps to run the project on your laptop.

WHAT YOU NEED
1) Python 3.10 or newer
2) Internet (for pip install and Supabase calls)

FOLDER TO OPEN
- Open the folder: SymptoCare

BACKEND (API)
Option A (easiest):
1) Double-click start_backend.bat
   - It creates .venv, installs packages, and starts the server.

Option B (manual):
1) Open PowerShell in the project folder
2) Run:
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r backend\requirements.txt
   python -m uvicorn backend.app:app --reload --host 127.0.0.1 --port 8000

BACKEND SETTINGS (SUPABASE)
Some features need Supabase (login, profile, history).
Create this file: backend\.env
Put these lines (replace with your values):
SUPABASE_URL=https://YOUR_PROJECT.supabase.co
SUPABASE_ANON_KEY=YOUR_ANON_KEY
CORS_ORIGINS=http://127.0.0.1:5500,http://localhost:5500

If you do not use Supabase, the prediction still works.

FRONTEND (WEB PAGE)
1) Open frontend\index.html in a browser
   - Or use VS Code "Live Server" extension.
2) The frontend calls the backend at:
   http://127.0.0.1:8000
   (You can change it in frontend\config.js)

QUICK CHECK
- Open this in the browser:
  http://127.0.0.1:8000/health
- You should see status "ok".

TROUBLESHOOT
- If "python" is not found, install Python and restart the terminal.
- If backend fails, delete .venv and run start_backend.bat again.
